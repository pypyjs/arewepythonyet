Are we Python yet?

Frankly, no, not yet – but we're working on it!

This site tracks performance metrics for PyPy.js, an in-browser version of the PyPy python interpreter powered by emscripten and asm.js.

There is currently no permanent build infrastructure. Benchmarks are run by hand and results are stored in static json files for offline processing. You can get the code from github if you'd like to run them locally, or just grab the raw data and play around with it.

Benchmarks and inspiration were taken from the PyPy speed center. Charts are powered by the wonderful MetricsGraphics.js.
